功能特色

🎙️

瀏覽器錄音

直接用瀏覽器錄音，支援即時麥克風偵測。錄音同時可顯示波形視覺化。

📝

即時語音辨識

若瀏覽器支援 Web Speech API，錄音同時即時產出逐字稿。也可手動編輯修正。

🤖

Whisper AI 轉錄

後端串接 OpenAI Whisper API，支援後端非同步轉錄，輸出高品質逐字稿。

📋

結構化 Meeting Minutes

自動整理：目標、執行摘要、討論重點、決策、行動項、風險、下一步。

💾

本地歷史管理

會議記錄與音檔綁定保存在瀏覽器 localStorage，可搜尋、載入、刪除。

📤

多格式匯出

支援匯出 Markdown、JSON、TXT，逐字稿與音檔皆可下載。

工作流程

1

錄音

輸入會議主題與與會人員，點擊錄音按鈕開始

2

逐字稿

瀏覽器即時辨識，或後送 Whisper API 處理

3

整理紀要

一鍵產生結構化 Meeting Minutes

4

匯出保存

下載 Markdown / JSON / TXT，含綁定音檔
