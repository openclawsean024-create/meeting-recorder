💡 自備 API Key，費用更省

Meeting Recorder 不在轉手收取 Whisper API 費用。你直接使用自己的 OpenAI API Key，按 OpenAI 官方費率計費（Whisper 約 NT$1.5/分鐘）。以 Free 方案 60 分鐘為例，Whisper 費用約 NT$90，相當划算。

在 API Key 設定頁面 填入你的 OpenAI API Key，即可開始使用。

常見問題

我真的需要自備 API Key 嗎？

是的。這是設計選擇，讓你完全掌控自己的資料與費用。我們不在中間抽成，Whisper API 費用由 OpenAI 直接向你收取。

我的錄音資料會被儲存嗎？

錄音檔在處理完成後即刪除。逐字稿與摘要儲存在你的帳戶下，可隨時刪除。

沒有 OpenAI API Key 怎麼辦？

可以前往 OpenAI Platform 申請，通常几分鐘即可完成。

用量用不完可以累積到下個月嗎？

目前方案為每月重新計算，不支援累積。未使用的分鐘數會在月底歸零。

如何升級或降級方案？

登入後可在「設定」頁面變更方案。變更會在下一個帳單週期生效。
